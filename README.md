# REM
REM is a memo application. This app is desktop application built with [TAURI](https://v2.tauri.app/).
- Frontend: React.js
- Backend: Rust

# Installation
1. 

# Develop build
```sh
cd src-tauri
npm run tauri dev
```

# Release build
```bash
npm run tauri build -- --bundles dmg
```
ref: https://tauri.app/distribute/dmg/
